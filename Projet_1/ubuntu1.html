Caractéristiques :

Unity est l'environnement de bureau par défaut de la variante principale d'Ubuntu. Il est l'interface préconisée et mise en avant par le projet Ubuntu.

Cependant, il existe des variantes d'Ubuntu pour chaque autres environements de bureau

  • Kubuntu pour KDE
  • Xubuntu pour XFCE
  • Lubuntu pout LXDE
  • etc...



Quelques variantes originales d'Ubuntu:

-Comme Ubuntu est OPEN SOURCE, la communauté a créé des variantes pour viser une certaine catégorie de personnes et ainsi élargir son nombre d'adeptes.



--> Pour l'éducation : Edubuntu

On trouve par exemple Edubuntu qui propose une variante officielle d'Ubuntu, conçue spécialement pour les écoles et l'enseignement. Edubuntu contient des applications utiles pour les classes dans les écoles primaires et secondaires, voire des niveaux supérieurs d'éducation.



--> Pour la création multimédia : Ubuntu Studio

Pour les artistes en herbe, on trouve parmis les variantes Ubuntu studio qui est une variante reconnue d'Ubuntu et axée sur la création numérique audio, vidéo ou graphique. Ubuntu studio utilise l'environnement de bureau léger XFCE et installe 2 noyaux différents : Le noyau generic et le noyau lowlatency permettant de traiter le signal audio avec une latence très faible.




--> Un enregistreur TV, Centre multimédia : Mythbuntu

Enfin nous trouvons parmis les variantes Mythbuntu (Myth tv).MythTV est un logiciel d'enregistrement numérique (télévision analogique, numérique, WebTV, etc.) extensible vers une station multimédia très complète via un système d'extensions (gestionnaire de photos, de vidéos, de musiques, lecture DVD & Blu-ray disk, navigation sur le web, etc.).
