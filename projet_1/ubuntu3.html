Tuto vidéo


Pré-Requis

--> Disposer de 10Go d'espace disque

--> Avoir minimum 1Go de mémoire vive

--> Être connecté à internet

--> Télécharger et installer Virtual-Box

--> Télécharger l'ISO d'Ubuntu



Installation

--> Lancer Virtual-Box

--> Créer une nouvelle machine

--> Donner pour Nom: ubuntu, Type: linux et Version: ubuntu(64bits)

--> Allouer une quantité de mémoire RAM de 2Go

--> Créer un disque virtuel

--> Choisir VDI comme type de fichiers disque dur

--> Choisir dynamiquement alloué pour le stockage sur disque dur

--> Saisir ubuntu pour le nom du fichier disque et donner 8Go en limite de stockage

--> Cliquer sur créer, puis lancer la nouvelle machine ubuntu

--> Saisir l'emplacement de l'ISO Ubuntu


--> Choisir Français et Installer Ubuntu

--> Cocher installer les mises à jours pendant l'installation

--> Effacer le disque et installer Ubuntu (Sauf pour le cas d'une installation physique,ou pour un dual boot)

--> Choisir Français pour la disposition du clavier

--> Une fois l'installation terminée à l'extinction de la machine aller dans la partie Stockage de la machine et vider le lecteur maitre IDE

--> Voilà
