Pourquoi ce thème ?

J’ai pris ce thème car il correspondait à mes attentes. Il reflete trés bien mon sujet qui est : " une école souhaite mettre en place un blog pour assurer la correspondance avec les familles au cours de cette semaine verte." Pour la recherche, j’ai simplement été sur wordpress.org/themes/ et dans recherche, School tout simplement, puis, je l’es télécharger en fichier zip. Une fois finis, je me suis connecter a mon serveur en SSH avec mon VPS, avec mon utilisateur root (ouvrir la fenêtre des documents, puis cliquez sur l’onglet fichier,se connecter au serveur). Une fois sur la page entrez votre adresse serveur, l’utilisateur et le mot de passe. Aller dans le chemin suivant : var/www/html/le_nom_de_votre_dossier_wordpress/wp_content/themes Déposé ici le fichier compressé zip du thème. Retourner sur votre wordpress en ligne et allez dans thème, le voilà installé, Il vous reste plus qu'à l'activer et à l'essayer.

Comment les installés ? Quels plugins je me suis approprié ?


La marche à suivre

Une fois vos plugins télécharger et décompreser, il vous suffis de le glisser dans le dossier suivant (var/www/html/le_nom_de_votre_dossier_wordpress/wp_content/plugins) et vous devez mettre le dossier a l’intérieur, il ne vous reste qu'à l’activer dans l’onglet pluging sur votre wordpress.


Un chat simple et éfficace

Pour mon sujet, j’avais besoin de faire un chat pour les parents et enseignants qui puissent communiquer entre eux et assurer une certaine correspondance. Ce plugin est très simple d'utilisation une fois activée. Il ne vous reste plus qu'à savoir ou vous voulez qu'il se situ sur vous wordpress.(Onglet/Apparence/Widget) plusieur option son possible cela dépend de votre thème. Pour ma part, il se situera, sur le sidebar Droit (Menu de droite)


Bip, bip, je suis Humain !

Même si vous n'avez jamais entendu ce mot, vous avez sûrement déjà croisé de nombreux captchas en surfant sur Internet. Un CAPTCHA est une suite de signes un peu déformés qu’on vous demande de recopier ou simplement un bouton à cliquer quand vous voulez vous enregistrer sur un site ou laisser un commentaire. Le but de l'opération : prouver que vous êtes bien un être humain et pas un programme informatique. Pour l'activer une fois installé : Onglet/Apparence/Widget. Il est complémentaire avec le plugin contact form 7, je vous parle juste après.


Une page contact

J’avais besoin de faire aussi une page contact, je vais vous apprendre à utiliser le plugin que j'ai pris. Une fois activée, dans votre menu à gauche se trouvera l'onglet contact cliquer dessus ensuite vous trouverez différente chose comme la configuration de base (cela dit vous pouvez toujours rajouter quelques champs à remplir en plus comme le tel, la date, une question et même le célèbre "reCAPTCHA" qui a été installé plus tôt ) Une fois que vous avez appliquer tous les champs que vous voulez que les personnes venant rentre il vous reste à copier le code indiqué au dessus [contact-form-7 id="162" title="Untitled"] dans une article ou une page et le tour est joué.


Vers un réseau social

J'ai aussi trouvé un Plugin Comments pour facebook.(J’ai pris une page d’une école). Le plugin Comments permet aux personnes de commenter le contenu de votre site à l’aide de leur compte Facebook. Les gens peuvent également partager leurs activités de commentaire avec leurs amis (et avec les amis de leurs amis) sur Facebook. Le plugin Comments propose également des outils de modération intégrés et un classement social par pertinence.

Le manège enchanté

J’avais besoin d’un élément présentant du contenu de façon ergonomique, sous la forme de diaporama avec ou sans navigation. On le retrouve sous le nom de slider, diaporama ou encore de carrousel. Pour l'activer une fois installé : Onglet/Apparence/Widget. Il vous reste plus qu'à importer vos média et configurer à votre guise
